This is a help section! 

ADD
Adds a to-do item. All the words behind add are entered as 1 to-do item to the list.
for example: node index.js add 'To do activity'

REMOVE
Removes a to-do item by its 1-base index.
for example: to remove item, execute: node index.js remove 'number of item'

RESET
Removes all to-do items from the list.
node index.js reset
